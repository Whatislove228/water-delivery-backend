# 🚰 Water Delivery Backend (Go)

Production-ready backend for a water delivery service.

Built with focus on:
- clean architecture
- scalability
- observability
- real-world business logic (orders, delivery slots, payments)

---

## 📦 Tech Stack

- **Go 1.26+**
- **Chi** - HTTP router
- **PostgreSQL 16**
- **Redis 7**
- **pgx** - DB driver
- **zap** - structured logging
- **Docker / Docker Compose**
